  • 请求地址：/user/tab/tabs/?

device_id Int Y 设备 ID 8800803362
iid Int Y 未知 14486549076

对于 device_id 和 iid 是如何生成的，目前未知。

  • 请求地址：/concern/v2/follow/my_follow/?

  • 请求参数

device_id Int Y 设备 ID 8800803362
iid Int N 未知 14486549076
